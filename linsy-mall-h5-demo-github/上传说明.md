# H5 商品详情 Demo — 上传 GitHub 说明

本文件夹为 **H5 详情页独立部署包**，不含 WEB 交易链路 Demo。

## 方式一：网页上传（推荐）

1. 登录 [GitHub](https://github.com) → **New repository**
2. 仓库名建议：`linsy-mall-h5-demo`（**Public**）
3. **不要**勾选 “Add a README”
4. 创建后 → **Add file → Upload files**
5. 上传本文件夹内**全部文件**（含 `.github` 文件夹）
6. Commit

### 开启 GitHub Pages

1. 仓库 **Settings → Pages**
2. **Build and deployment → Source** 选 **GitHub Actions**
3. 等待 Actions「Deploy GitHub Pages」完成（约 1–2 分钟）
4. 访问：

```
https://<你的用户名>.github.io/linsy-mall-h5-demo/
```

手机浏览器打开即可预览 H5 效果。
